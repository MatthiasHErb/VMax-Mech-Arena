# Mech Arena — Kampfroboter

Ein Browser-Spiel im Stil von Mech Arena: Roboter bauen, in der Arena kämpfen, Credits verdienen und den Mech verbessern.

## Spielstart

Öffne `index.html` im Browser (Doppelklick oder z.B. mit Live-Server).

## Ablauf

1. **Garage** — Deinen Mech ansehen und ausrüsten. Du startest mit Scout-Chassis, Leichter Kanone und Leichtpanzerung. Stats (Leben, Rüstung, Schaden, Geschwindigkeit) werden aus den ausgerüsteten Teilen berechnet. Waffe links und rechts können separat gewählt werden.

2. **Arena** — Einen Gegner wählen und „Kampf starten“. Der Kampf ist rundenbasiert; Geschwindigkeit entscheidet, wer zuerst zuschlägt. Sieg bringt Credits.

3. **Shop** — Mit Credits neue Teile kaufen (Chassis, Waffen, Rüstung). Gekaufte Teile kannst du in der Garage ausrüsten.

## Technik

- Nur HTML, CSS und JavaScript, keine Installation nötig.
- Schriftarten: Orbitron, Rajdhani (Google Fonts).

Viel Spaß beim Bauen und Kämpfen.
